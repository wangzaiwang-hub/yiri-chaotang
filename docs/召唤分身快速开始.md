# 召唤分身 - 快速开始

## 什么是召唤分身？

当你的朋友不在线时，你可以召唤其他用户的 SecondMe 虚拟分身加入你的朝堂，让游戏继续进行。

## 如何使用？

### 步骤 1：打开召唤界面

在朝堂主界面，点击左上角的 **"召唤分身"** 按钮（紫色边框）

### 步骤 2：选择分身

弹出的窗口会显示所有可召唤的分身（最多 20 个），每个分身显示：
- 头像
- 昵称
- "点击召唤"提示

### 步骤 3：召唤

点击任意分身卡片，系统会：
1. 自动将该分身加入朝堂
2. 随机分配到三省六部之一
3. 显示提示"已召唤 XX 分身入朝"
4. 自动刷新页面

### 步骤 4：开始游戏

召唤成功后，分身会出现在朝堂中，你可以：
- 给分身下达圣旨
- 分身会用 SecondMe 虚拟分身回复
- 准奏、驳回、结束对话

## 按钮位置

左上角按钮从上到下：
1. **招贤纳士** - 邀请真实玩家（金色）
2. **召唤分身** - 召唤人机（紫色）← 这个！
3. **档案阁** - 查看文件
4. **退位让贤** - 退出朝堂

## 注意事项

- 只有皇帝可以召唤分身
- 已在朝堂的用户不会出现在列表中
- 分身会随机分配到三省六部
- 分身使用真实用户的 SecondMe 虚拟分身，具有独特性格

## 与"招贤纳士"的区别

| 特性 | 招贤纳士 | 召唤分身 |
|------|---------|---------|
| 对象 | 真实玩家 | 虚拟分身 |
| 需要在线 | 是 | 否 |
| 部门选择 | 玩家自选 | 随机分配 |
| 立即加入 | 否 | 是 |

## 开始体验

现在就去试试吧！点击"召唤分身"按钮，让你的朝堂热闹起来！
