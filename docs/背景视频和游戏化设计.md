# 背景视频和游戏化设计

## 背景视频实现

### 视频背景
- 使用 `bg.mp4` 作为主界面的动态背景
- 视频自动播放、循环、静音
- 如果视频加载失败，自动降级到静态图片 `bg.png`
- 添加半透明黑色遮罩（10% 不透明度）让前景内容更清晰

### 实现细节
```tsx
<video
  autoPlay
  loop
  muted
  playsInline
  className="absolute inset-0 w-full h-full object-cover"
  style={{ zIndex: -1 }}
>
  <source src={bgVideo} type="video/mp4" />
</video>
```

## 游戏化字体设计

### 中文字体
使用 Google Fonts 的三种中文字体：

1. **Noto Serif SC（思源宋体）**
   - 优雅的衬线字体
   - 用于正文和说明文字
   - 类名：`.text-chinese-elegant`

2. **Ma Shan Zheng（马善政楷体）**
   - 手写风格的书法字体
   - 用于标题和重要文字
   - 类名：`.text-chinese-title`

3. **ZCOOL XiaoWei（站酷小薇体）**
   - 现代中文字体
   - 作为备用字体

### 特殊文字效果
- `.text-gold-gradient` - 金色渐变文字效果
- 字间距调整（`letter-spacing`）
- 文字阴影（`text-shadow`）

## 动画效果

### 入场动画
1. **fadeInUp** - 淡入上升
   - 用于：怨气值面板
   - 效果：从下方淡入并上升

2. **scaleIn** - 缩放淡入
   - 用于：性别选择弹窗
   - 效果：从小到大缩放并淡入

3. **slideInLeft** - 从左滑入
   - 用于：左上角按钮组
   - 效果：从左侧滑入，带延迟效果

### 循环动画
1. **float** - 漂浮动画
   - 用于：皇上/皇后图片
   - 效果：上下缓慢漂浮

2. **glow** - 发光脉冲
   - 用于：选中的选项
   - 效果：光晕脉冲效果

3. **pulse-glow** - 脉冲发光
   - 用于：主要操作按钮
   - 效果：持续的光晕脉冲

4. **shimmer** - 闪光效果
   - 用于：特殊按钮
   - 效果：光线扫过效果

### 交互动画
1. **btn-imperial** - 皇家按钮效果
   - 悬停时：波纹扩散效果
   - 点击时：缩小效果（scale 0.95）

2. **bubble-effect** - 气泡效果
   - 悬停时：显示光晕边框

## 视觉层次

### Z-Index 层级
- `-1`: 背景视频
- `0`: 半透明遮罩
- `1`: 主要内容层
- `10`: 左上角按钮、右上角怨气值
- `20`: 皇帝气泡
- `30`: 大臣气泡
- `40`: thinking/idea GIF
- `50`: 审批按钮和遮罩
- `100`: 弹窗（准奏、驳回）
- `200`: 性别选择弹窗

### 颜色方案
- 主色调：琥珀色/金色（amber）
- 强调色：粉色（皇后）、红色（驳回/退出）
- 背景：白色半透明 + 毛玻璃效果
- 文字：深琥珀色（amber-900）

## 性能优化

### 视频优化
- 使用 `playsInline` 属性确保移动端正常播放
- 静音播放避免自动播放限制
- 提供降级方案（静态图片）

### 动画优化
- 使用 CSS 动画而非 JavaScript
- 使用 `transform` 和 `opacity` 实现动画（GPU 加速）
- 合理使用 `animation-delay` 避免同时触发

### 字体优化
- 使用 Google Fonts CDN
- 设置 `display=swap` 避免字体加载阻塞
- 提供系统字体作为降级方案

## 使用示例

### 标题文字
```tsx
<h1 className="text-chinese-title text-gold-gradient">
  欢迎来到朝堂
</h1>
```

### 正文文字
```tsx
<p className="text-chinese-elegant">
  你还没有加入任何朝堂，创建一个开始游戏吧！
</p>
```

### 动画按钮
```tsx
<button className="btn-imperial animate-pulse-glow text-chinese-elegant">
  创建朝堂
</button>
```

### 漂浮图片
```tsx
<img className="animate-float" src={kingImage} alt="皇上" />
```

## 浏览器兼容性

- 视频背景：支持所有现代浏览器
- CSS 动画：支持所有现代浏览器
- 毛玻璃效果：支持 Chrome 76+, Safari 9+, Firefox 103+
- 中文字体：通过 Google Fonts CDN 加载，全平台支持
