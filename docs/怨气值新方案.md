# 怨气值新方案 - 真人 + 人机混合

## 核心理念

**怨气值是大臣的"不满度"，用来驱动游戏事件和解锁功能**

- 怨气值不是用来反叛的，而是用来**激励大臣做出有趣的事**
- 真人大臣和人机大臣都能参与
- 怨气值越高 → 大臣越"闹腾" → 游戏越有趣

---

## 怨气值的作用

### 1. 解锁大臣的特殊行为

**怨气值 0-20：安分守己**
- 正常完成任务
- 偶尔吐槽（自动生成）
- 没有特殊行为

**怨气值 20-50：开始闹腾**
- 🎭 自动发布"无聊提示语"（吐槽皇帝）
- 💬 在悄悄话中吐槽（真人大臣可以手动发送）
- 🎪 要求"娱乐任务"（唱歌、跳舞、讲笑话）
- 📢 在朝堂中发表"怨言"

**怨气值 50-80：开始捣乱**
- 🚫 有概率"偷懒"（任务完成质量下降）
- 😤 发送"激烈吐槽"
- 🎯 要求"特殊奖励"（加薪、升官）
- 📣 在朝堂中发表"激烈言论"

**怨气值 80-100：反叛边缘**
- ⚠️ 有概率"罢工"（拒绝执行任务）
- 🔥 发送"极端吐槽"
- 💰 要求"巨额赔偿"
- 📢 在朝堂中发表"反叛言论"
- 🎪 可能"自动请假"（3天无法操作）

```typescript
const grudgeGain = {
```

## 怨气值驱动的事件

### 自动生成的吐槽（人机大臣）

**怨气值 20+：**
```
大臣A 在朝堂中吐槽：
"皇帝啊，你这样压榨我们，我们怎么活啊？"
"我已经连续 3 天没有休息了！"
"这任务质量要求这么高，你自己去做啊！"
```

**怨气值 50+：**
```
大臣A 在朝堂中激烈言论：
"我受够了！这样下去我要罢工了！"
"皇帝，你是想把我们都逼反吗？"
"我要求加薪 50%，否则我就请假！"
```

**怨气值 80+：**
```
大臣A 在朝堂中反叛言论：
"各位，咱们一起罢工吧！"
"这个皇帝不配统治我们！"
"我要请假 3 天，谁敢拦我？"
```

### 自动行为（人机大臣）

**怨气值 50+：**
- 有 30% 概率"偷懒"
  - 任务完成，但质量 -30%
  - 皇帝审批时会看到"质量不达标"

**怨气值 80+：**
- 有 50% 概率"罢工"
  - 拒绝执行任务
  - 皇帝看到"大臣拒绝执行"
  - 怨气值 +10（更加不满）

### 真人大臣的互动

**真人大臣可以：**
1. 发送吐槽悄悄话（增加怨气）
2. 发送要求悄悄话（要求奖励）
3. 在朝堂中发表言论（公开表达不满）
4. 选择"罢工"（拒绝执行任务）


### 3. 处理罢工

**当大臣罢工时：**

```
大臣A 拒绝执行任务

皇帝的选择：
[1] 强制执行 (消耗 50 幸福度)
    → 任务完成，但质量 -50%
    → 大臣怨气 +30（更加不满）

[2] 接受罢工 (消耗 0 幸福度)
    → 任务失败
    → 大臣怨气 -5（释放了不满）
    → 幸福度 -10（任务失败）

[3] 协商 (消耗 30 幸福度)
    → 大臣同意执行
    → 大臣怨气 -10
    → 任务正常完成
```

## 怨气值的平衡

```typescript
// 每天的怨气值变化
const dailyGrudgeFlow = {
  // 正常情况
  taskCompleted: -5,             // 完成任务
  dailyDecay: -3,                // 时间衰减
  noTask: +8,                    // 没有任务
  
  // 不好的情况
  taskRejected: +15,             // 任务被打回
  punishment: +20,               // 被惩罚
  
  // 平衡点：
  // 如果皇帝不管：每天 +8-15 怨气
  // 需要 5-10 天达到反叛阈值（80分）
  // 皇帝有足够时间反应
};
```

## 这个方案的优势

1. **真人 + 人机都能参与**
   - 真人大臣可以主动发送悄悄话
   - 人机大臣自动生成吐槽和行为

2. **怨气值驱动游戏事件**
   - 不是反叛，而是"闹腾"
   - 增加游戏的趣味性和挑战性

3. **皇帝有明确的应对机制**
   - 可以通过赏赐、升官来安抚
   - 需要平衡幸福度消耗和怨气值管理

4. **自然的游戏循环**
   - 怨气值越高 → 大臣越闹腾
   - 皇帝需要主动管理 → 消耗幸福度
   - 形成"压迫-安抚"的动态平衡

5. **高度的可玩性**
   - 每个大臣都有独立的怨气值
   - 皇帝需要区别对待
   - 真人大臣可以主动参与
