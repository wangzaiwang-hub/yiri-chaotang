## 🚀 现在你可以

### 1. 测试登录流程

1. 访问：http://localhost:3000
2. 点击"使用 SecondMe 登录"
3. 会跳转到 SecondMe 授权页面
4. 使用你的 SecondMe 账号登录
5. 授权后会自动跳转回应用

### 2. 创建朝堂

登录成功后：
1. 点击"创建朝堂"
2. 填写朝堂名称和描述
3. 创建成功

### 3. 邀请好友

1. 分享邀请链接给好友
2. 好友也需要用 SecondMe 登录
3. 好友加入后可以开始游戏

### 4. 发布任务

1. 进入"帝王宝座"
2. 填写任务信息
3. 选择臣子（好友的虚拟分身）
4. 发布圣旨

### 5. 虚拟分身执行任务

1. 臣子的虚拟分身会自动执行任务
2. 使用 SecondMe Chat API 生成回复
3. 实时流式显示执行过程

## 🎮 游戏流程

### 完整游戏循环

1. **创建朝堂** → 邀请 4-10 个好友
2. **第一天** → 创建者自动成为皇帝
3. **发布任务** → 皇帝给臣子的虚拟分身发任务
4. **执行任务** → 虚拟分身自动执行（调用 SecondMe API）
5. **审批任务** → 皇帝批准或打回
6. **累计怨气** → 完成任务增加怨气值
7. **每日结算** → 凌晨 00:00 自动结算
8. **皇帝轮换** → 怨气最高者登基
9. **复仇时刻** → 新皇帝可以报复昨日的皇帝
10. **循环往复** → 每天轮换，永不停歇

## 🐛 遇到问题？

### 后端无法启动
- 检查 `backend/.env` 文件是否正确配置
- 确保 Supabase 密钥正确
- 查看错误日志

### 登录失败
- 确保 SecondMe OAuth2 配置正确
- 检查回调 URL 是否匹配
- 查看浏览器控制台错误

### 任务执行失败
- 确保用户已授权 SecondMe
- 检查 Token 是否过期
- 查看后端日志
