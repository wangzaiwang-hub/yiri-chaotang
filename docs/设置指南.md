# 🔧 环境配置向导

## 当前问题

❌ 后端服务未启动（缺少环境变量）  
❌ 点击登录无法跳转到 SecondMe

## 解决方案

### 方案 1：完整配置（推荐，可以正常使用）

#### 步骤 1：配置 Supabase

1. 访问 https://supabase.com
2. 点击 "New Project"
3. 填写项目信息并创建
4. 等待项目创建完成（约 2 分钟）
5. 进入项目后，点击左侧 "SQL Editor"
6. 点击 "New Query"
7. 复制 `database/init.sql` 的全部内容并粘贴
8. 点击 "Run" 执行
9. 进入 Settings > API，复制以下信息：
   - URL: `https://xxxxx.supabase.co`
   - anon public: `eyJhbGc...`（很长的字符串）
   - service_role: `eyJhbGc...`（很长的字符串，保密）

#### 步骤 2：配置 SecondMe

1. 访问 https://develop.second-me.cn/integrations/list
2. 使用 SecondMe 账号登录（如果没有账号需要先注册）
3. 点击"创建应用"或"新建应用"
4. 填写应用信息：
   ```
   应用名称：朝堂
   应用描述：虚拟分身权力游戏
   回调 URL：http://localhost:3001/api/auth/callback
   权限范围：勾选 userinfo, chat.write, chat.read
   ```
5. 创建后会显示：
   - Client ID: `9b8f2a1b-905d-48c3-8db3...`
   - Client Secret: `0c1eba70c0cf279ae209be...`（保密）

#### 步骤 3：配置环境变量

编辑 `backend/.env` 文件：

```env
NODE_ENV=development
PORT=3001

# Supabase（从步骤1获取）
SUPABASE_URL=https://xxxxx.supabase.co
SUPABASE_ANON_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...
SUPABASE_SERVICE_KEY=eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

# SecondMe（从步骤2获取）
SECONDME_CLIENT_ID=9b8f2a1b-905d-48c3-8db3...
SECONDME_CLIENT_SECRET=0c1eba70c0cf279ae209be...
BASE_URL=http://localhost:3001
FRONTEND_URL=http://localhost:3000

# OpenAI (可选，暂时不需要)
# OPENAI_API_KEY=sk-your-openai-key

# JWT
JWT_SECRET=my-super-secret-jwt-key-change-this-in-production
```

#### 步骤 4：重启后端

```bash
# 如果后端正在运行，先停止（Ctrl+C）
cd backend
npm run dev
```

#### 步骤 5：测试登录

1. 访问 http://localhost:3000
2. 点击"使用 SecondMe 登录"
3. 应该会跳转到 SecondMe 授权页面
4. 登录并授权后会跳转回应用

### 方案 2：快速演示模式（仅用于测试前端界面）

如果你暂时不想配置 Supabase 和 SecondMe，可以使用演示模式：

#### 修改前端代码，添加演示登录

编辑 `frontend/src/pages/Login.tsx`，在 `handleLogin` 函数中添加：

```typescript
const handleLogin = () => {
  // 演示模式：直接模拟登录成功
  if (confirm('演示模式：是否模拟登录成功？\n（需要真实登录请先配置后端环境变量）')) {
    const demoUser = {
      id: 'demo-user-123',
      nickname: '演示用户',
      avatar: 'https://api.dicebear.com/7.x/avataaars/svg?seed=demo',
      bio: '这是演示账号'
    };
    const demoToken = 'demo-token-' + Date.now();
    setAuth(demoUser, demoToken);
    navigate('/');
    return;
  }
  
  // 正常登录流程
  authAPI.login();
};
```

## 常见问题

### Q: Supabase 创建项目很慢？
A: 是的，通常需要 1-2 分钟，请耐心等待

### Q: SecondMe 注册需要什么？
A: 需要手机号或邮箱，可以使用微信登录

### Q: 环境变量配置错误会怎样？
A: 后端会启动失败并显示错误信息，根据错误提示修改即可

### Q: 可以跳过 OpenAI 配置吗？
A: 可以，OpenAI 仅用于生成虚拟惩罚内容，不影响核心功能
