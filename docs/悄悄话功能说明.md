# 悄悄话功能说明

## 功能概述
大臣之间可以互相说悄悄话，但这些悄悄话也会被皇上看到。大臣们会轻松幽默地吐槽皇上、调侃朝堂趣事。

## 数据库设计

### whispers 表
```sql
- id: UUID (主键)
- court_id: UUID (朝堂 ID)
- from_user_id: UUID (发送者 ID)
- to_user_id: UUID (接收者 ID)
- content: TEXT (悄悄话内容，AI 生成)
- created_at: TIMESTAMP (创建时间)
- read_by_emperor: BOOLEAN (皇帝是否已读)
- read_by_recipient: BOOLEAN (接收者是否已读)
```

## 后端 API

### 1. 发送悄悄话
**接口**: `POST /api/whispers/send`

**参数**:
```json
{
  "court_id": "朝堂ID",
  "from_user_id": "发送者ID",
  "to_user_id": "接收者ID",
  "topic": "话题描述"
}
```

**功能**:
- 使用 AI 根据话题生成轻松幽默的悄悄话内容
- 可以适当吐槽皇上，但不会太过分
- 使用古风口语化风格
- 字数控制在 50-100 字

**AI Prompt 特点**:
- 明确这是游戏角色扮演
- 可以取笑皇上，但皇上能看到
- 轻松幽默，带点调侃
- 古风口语化，不要太文言文

### 2. 获取悄悄话列表
**接口**: `GET /api/whispers?court_id=xxx&user_id=xxx`

**权限**:
- 皇帝：可以看到所有悄悄话
- 大臣：只能看到与自己相关的悄悄话（发送或接收）

**返回**: 最近 50 条悄悄话，按时间倒序

### 3. 标记为已读
**接口**: `POST /api/whispers/:id/read`

**参数**:
```json
{
  "user_id": "用户ID",
  "is_emperor": true/false
}
```

## 前端 API 方法

```typescript
// 发送悄悄话
whisperAPI.send(courtId, fromUserId, toUserId, topic)

// 获取悄悄话列表
whisperAPI.list(courtId, userId)

// 标记为已读
whisperAPI.markAsRead(whisperId, userId, isEmperor)
```

## 使用场景

### 大臣视角
1. 点击其他大臣头像
2. 选择"说悄悄话"
3. 输入话题（如：今天的任务、皇上的决定等）
4. AI 自动生成幽默的悄悄话
5. 可以查看与自己相关的悄悄话

### 皇帝视角
1. 可以偷听所有大臣之间的悄悄话
2. 看到大臣们吐槽自己
3. 增加游戏趣味性和互动性

## AI 生成示例

**话题**: "今天的任务太难了"

**生成内容**:
```
哎呀老张，你说咱们皇上今天又下了个什么奇怪的任务，
我看着都头疼。不过咱们还是得好好干活，
万一被听到就惨了，哈哈！
```

**话题**: "皇上最近心情不好"

**生成内容**:
```
小李啊，最近皇上好像有点不高兴，
咱们办事可得小心点。不过说实话，
有时候皇上的想法确实挺有意思的，哈哈。
```

## 下一步开发

### 前端 UI 需要实现：
1. **悄悄话入口**：
   - 在大臣头像上添加"说悄悄话"按钮
   - 或在怨气值面板添加悄悄话图标

2. **悄悄话弹窗**：
   - 选择接收者（其他大臣）
   - 输入话题
   - 显示 AI 生成的内容
   - 发送确认

3. **悄悄话列表**：
   - 显示最近的悄悄话
   - 区分发送/接收
   - 皇帝视角显示所有悄悄话
   - 未读标记

4. **悄悄话气泡**：
   - 类似任务气泡的样式
   - 显示在大臣之间
   - 皇帝可以看到半透明的"偷听"效果

## 技术要点

1. **WebSocket 实时推送**：
   - 新悄悄话通过 WebSocket 广播
   - 实时更新列表

2. **权限控制**：
   - 大臣只能看到与自己相关的
   - 皇帝可以看到所有

3. **AI 生成**：
   - 轻松幽默的风格
   - 适度吐槽，不会太过分
   - 古风口语化

## 注意事项

1. 悄悄话内容由 AI 生成，确保游戏性和趣味性
2. 皇帝能看到所有悄悄话，增加游戏的戏剧性
3. 大臣之间的互动增加朝堂的活跃度
4. 内容轻松幽默，不会影响游戏体验
