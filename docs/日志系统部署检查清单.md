### 常见问题

**Q: 日志没有显示？**
A: 检查是否有未查看日志，查看浏览器控制台是否有错误

**Q: 日志显示但无法确认？**
A: 检查是否查看了所有日志，确认按钮只在查看完所有日志后启用

**Q: 数据库错误？**
A: 检查数据库连接，查看后端日志

**Q: 性能问题？**
A: 检查数据库索引，查看查询性能

### 紧急回滚

如果出现严重问题：

```bash
# 1. 停止后端服务
systemctl stop backend

# 2. 回滚数据库（如需要）
# 恢复备份或执行回滚脚本

# 3. 回滚前端（如需要）
# 部署上一个版本

# 4. 重启服务
systemctl start backend
```
