# 疆土拓展功能说明

## 功能概述
实现了一个"疆土拓展"功能，在页面顶部显示一个进度条，点击进度条打开疆土拓展弹窗，允许用户通过翻卡牌的方式选择不同的场景背景，切换朝堂的视觉风格。

## 已实现的功能

### 1. 顶部进度条
- **位置**：页面最顶部，横跨整个屏幕
- **样式**：
  - 深色半透明背景（琥珀色/橙色渐变）
  - 金色边框，hover 时高亮
  - 显示"疆土拓展"标题和当前场景名称
  - 进度条显示"百姓幸福度：60%"
  - 进度条内有闪光动画效果
- **交互**：点击整个进度条区域打开疆土拓展弹窗
- **响应式**：自动适配屏幕宽度

### 2. 场景资源
已导入 6 个国家场景图片：
- **华夏** (huaxia.jpg) - 中原大地，繁华盛世
- **东瀛** (dongying.jpg) - 海岛之国，樱花烂漫
- **高丽** (gaoli.jpg) - 半岛之邦，文化璀璨
- **安南** (annan.jpeg) - 南国风情，物产丰饶
- **暹罗** (xianluo.jpeg) - 热带王国，佛塔林立
- **大食** (dashi.jpeg) - 西域明珠，商贾云集

### 3. UI 组件

#### 疆土拓展弹窗
- 显示所有可用场景的卡牌（背面）
- 3 列网格布局
- 当前使用的场景会显示"当前"标签
- 点击卡牌触发翻牌动画

#### 场景翻牌动画
- 全屏遮罩，黑色半透明背景
- 卡牌 3D 翻转效果（1 秒动画）
- 翻转后显示场景图片、名称、描述
- 提供"取消"和"选择此场景"按钮

- 主界面背景图片动态切换
- 使用 `getCurrentBackgroundImage()` 函数获取当前场景图片

### 6. 布局调整
- 左上角按钮组下移至 `top-20`（为进度条留出空间）
- 右上角"重生转世"按钮下移至 `top-20`
- 进度条固定在 `top-0`，z-index 为 20

## 使用流程

1. 用户看到页面顶部的进度条，显示当前幸福度和场景名称
2. 点击进度条任意位置
3. 弹出疆土拓展弹窗，显示所有场景卡牌（背面）
4. 用户点击任意卡牌
5. 卡牌翻转，显示场景图片和信息
6. 用户点击"选择此场景"确认切换
7. 背景图片立即切换，显示成功提示
8. 选择保存到 localStorage

## 技术细节

### 进度条设计
- 使用渐变背景：`from-amber-900/80 to-orange-900/80`
- 进度填充：`from-yellow-500 via-amber-500 to-orange-500`
- 闪光动画：使用 CSS `@keyframes shimmer`
- 响应式文字：使用 `text-chinese-elegant` 类
- Hover 效果：边框颜色从 `yellow-600/50` 变为 `yellow-400`

### 动画效果
- **翻牌动画**：
  - 使用 CSS 3D transform 实现
  - `perspective: 1000px` 创建 3D 空间
  - `transform: rotateY(180deg)` 实现翻转
  - `backfaceVisibility: hidden` 隐藏背面
  - 动画时长：1 秒

- **闪光动画**：
  - `@keyframes shimmer` 从左到右移动
  - 2 秒循环
  - 白色半透明渐变

### 样式设计
- 复用召唤分身的卡牌样式
- 金色/琥珀色主题，符合古风设计
- 场景卡片显示渐变遮罩，突出文字信息
- 响应式布局，适配不同屏幕尺寸
