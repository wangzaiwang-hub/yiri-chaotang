### 唯一缺少的：Supabase 数据库配置

需要你完成以下 3 个步骤（5分钟）：

#### 步骤 1：创建 Supabase 项目
1. 访问：https://supabase.com
2. 点击 "Start your project" 或 "New Project"
3. 使用 GitHub 登录（最快）
4. 创建项目：
   - Name: `chaotang`
   - Password: 设置一个密码
   - Region: `Northeast Asia (Seoul)`
5. 等待 1-2 分钟

#### 步骤 2：初始化数据库
1. 在 Supabase 项目中，点击左侧 "SQL Editor"
2. 点击 "New Query"
3. 复制 `database/init.sql` 的全部内容
4. 粘贴并点击 "Run"

#### 步骤 3：获取密钥并配置
1. 点击 Settings > API
2. 复制三个值：
   - Project URL
   - anon public
   - service_role

3. 编辑 `backend/.env` 文件，替换这三行：
```env
SUPABASE_URL=你的Project URL
SUPABASE_ANON_KEY=你的anon public
SUPABASE_SERVICE_KEY=你的service_role
```
