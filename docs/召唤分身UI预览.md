# 召唤分身 - UI 预览

## 按钮样式

### 位置
左上角按钮组，第二个位置（在"招贤纳士"和"档案阁"之间）

### 视觉效果
```
┌─────────────────┐
│  招贤纳士  🏛️   │ ← 金色边框
├─────────────────┤
│  召唤分身  ✨   │ ← 紫色边框（新增）
├─────────────────┤
│  档案阁    📜   │ ← 金色边框
├─────────────────┤
│  退位让贤  👑   │ ← 红色文字
└─────────────────┘
```

### CSS 样式
- 背景：`bg-white/80` - 白色半透明
- 边框：`border-purple-400` - 紫色边框
- 文字：`text-purple-700` - 紫色文字
- 圆角：`rounded-2xl` - 大圆角
- 尺寸：`px-5 py-2` - 适中内边距
- 阴影：`shadow-lg` - 大阴影
- 动画：`animate-slideInLeft` - 从左滑入

## 分身卡片样式

### 单个卡片
```
┌─────────────────┐
│                 │
│      👤         │ ← 头像（圆形，20x20）
│                 │
│    张三         │ ← 昵称（紫色粗体）
│                 │
│  点击召唤       │ ← 提示（小字）
│                 │
└─────────────────┘
```

### 卡片特性
- 背景：白色 (`bg-white`)
- 边框：2px 紫色 (`border-2 border-purple-300`)
- 圆角：大圆角 (`rounded-2xl`)
- 内边距：`p-6`
- 悬停效果：
  - 背景变浅紫色 (`hover:bg-purple-50`)
  - 边框变深紫色 (`hover:border-purple-500`)
  - 放大 1.05 倍 (`hover:scale-105`)
  - 阴影加深 (`hover:shadow-xl`)

### 头像样式
- 尺寸：`w-20 h-20` (80x80px)
- 形状：圆形 (`rounded-full`)
- 边框：2px 紫色 (`border-2 border-purple-300`)
- 如果没有头像：显示渐变背景 + 首字母

## Toast 提示

召唤成功时：
```
┌─────────────────────────────────┐
│  ✓  已召唤 张三 分身入朝        │
└─────────────────────────────────┘
```

- 位置：顶部居中
- 背景：金色渐变 (`from-amber-400 to-amber-500`)
- 边框：4px 金色 (`border-amber-600`)
- 文字：金色 (`text-amber-900`)
- 持续时间：3 秒
- 动画：缩放进入 (`animate-scaleIn`)

## 颜色方案

### 主色调
- 紫色：`purple-400`, `purple-500`, `purple-700`, `purple-900`
- 粉色：`pink-50`, `pink-400`
- 白色：`white`, `white/80`, `white/90`

### 对比色
- 金色：用于成功提示
- 红色：用于错误提示
- 灰色：用于取消按钮

## 响应式设计

- 桌面端：3 列网格
- 平板端：2 列网格（可选优化）
- 移动端：1 列网格（可选优化）

## 动画效果

1. **按钮进入**：从左滑入 (`animate-slideInLeft`)
2. **弹窗进入**：缩放进入 (`animate-scaleIn`)
3. **卡片悬停**：放大 + 阴影
4. **Toast 提示**：缩放进入 + 淡出

## 交互反馈

1. **点击按钮**：弹窗立即出现
2. **悬停卡片**：视觉反馈（放大、变色）
3. **点击卡片**：禁用状态 + 加载提示
4. **召唤成功**：Toast 提示 + 自动刷新
5. **召唤失败**：错误 Toast 提示

## 可访问性

- 所有按钮都有清晰的文字标签
- 禁用状态有视觉反馈（透明度降低）
- 关闭按钮明显且易于点击
- 卡片有足够的点击区域
