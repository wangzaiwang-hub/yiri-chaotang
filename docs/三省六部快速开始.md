## 第二步：重启后端

```bash
cd backend
npm run dev
```

## 第三步：重启前端

```bash
cd frontend
npm run dev
```

## 第四步：体验三省六部

1. 登录系统
2. 创建朝堂（你将成为皇帝）
3. 点击"邀请好友"生成邀请链接
4. 好友通过链接加入，选择部门（中书省、门下省、尚书省或六部之一）
5. 皇帝下旨创建任务
6. 点击"📋 任务看板"查看任务
7. 各部门按流程处理任务：
   - 中书省：规划任务
   - 门下省：审核方案
   - 尚书省：派发任务
   - 六部：执行任务
   - 尚书省：汇总回奏

## 任务流转示例

```
皇帝下旨："开发一个用户注册系统"
  ↓
中书省规划：拆解为数据库设计、API开发、前端界面、测试等子任务
  ↓
门下省审核：检查方案是否完整，准奏或封驳
  ↓
尚书省派发：分配给各部门
  ↓
六部执行：
  - 兵部：开发 API
  - 工部：部署数据库
  - 礼部：编写文档
  - 刑部：安全审计
  ↓
尚书省汇总：形成奏折回报皇帝
```

完整文档请查看 `docs/三省六部实施指南.md`
