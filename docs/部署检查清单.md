## 后端部署（Railway）

### 1. 准备工作
```bash
# 安装 Railway CLI
npm install -g @railway/cli

# 登录
railway login
```

### 2. 创建项目
```bash
cd backend
railway init
```

### 3. 配置环境变量
在 Railway Dashboard 中添加：
- [ ] NODE_ENV=production
- [ ] PORT=3001
- [ ] SUPABASE_URL
- [ ] SUPABASE_ANON_KEY
- [ ] SUPABASE_SERVICE_KEY
- [ ] SECONDME_CLIENT_ID
- [ ] SECONDME_CLIENT_SECRET
- [ ] BASE_URL（生产环境 URL）
- [ ] FRONTEND_URL（前端 URL）
- [ ] OPENAI_API_KEY（可选）
- [ ] JWT_SECRET

### 4. 部署
```bash
railway up
```

### 5. 验证
- [ ] 访问 /health 端点
- [ ] 测试 OAuth2 登录
- [ ] 测试 API 接口
- [ ] 查看日志

## 前端部署（Vercel）

### 1. 准备工作
```bash
# 安装 Vercel CLI
npm install -g vercel

# 登录
vercel login
```

### 2. 配置环境变量
在 Vercel Dashboard 中添加：
- [ ] VITE_API_BASE_URL（后端 URL）
- [ ] VITE_SECONDME_CLIENT_ID

### 3. 部署
```bash
cd frontend
vercel --prod
```

### 4. 验证
