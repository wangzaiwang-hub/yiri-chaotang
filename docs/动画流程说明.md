# 动画流程说明

## 完整的交互流程

### 1. 发送任务

**流程**：
1. 用户点击"下达圣旨"
2. 立即进入 `emperor_bubble` 阶段
3. 显示皇帝气泡，打字机效果显示任务描述
4. 打字完成后等待 2 秒
5. 气泡淡出（1 秒）
6. 进入 `waiting_ai` 阶段，显示 `thinking.gif`

**状态变化**：
- `animationPhase`: `done` → `emperor_bubble` → `waiting_ai`
- `thinkingState`: `none` → `none` → `thinking`
- `bubbleVisible`: `false` → `true` → `false`

### 2. AI 回复到达

**流程**：
1. 后端返回 AI 回复，任务的 `result` 字段更新
2. 检测到 `animationPhase === 'waiting_ai'` 且有 `result`
3. 切换到 `idea.gif`，进入 `ai_ready` 阶段
4. 显示 `idea.gif` 1 秒
5. 进入 `minister_bubble` 阶段，显示大臣气泡
6. 打字机效果显示大臣回复
7. 打字完成后播放 TTS 语音
8. 等待 3 秒后气泡淡出（1 秒）
9. 同时显示结准驳按钮和遮罩
10. 进入 `done` 阶段

**状态变化**：
- `animationPhase`: `waiting_ai` → `ai_ready` → `minister_bubble` → `done`
- `thinkingState`: `thinking` → `idea` → `none`
- `bubbleVisible`: `false` → `false` → `true` → `false`
- `showApprovalButtons`: `false` → `true`

### 3. 准奏（批准）

**流程**：
1. 用户点击"准"按钮，可选填写补充说明
2. 隐藏审批按钮和遮罩
3. 进入 `waiting_ai` 阶段，显示 `thinking.gif`
4. 后续流程同"AI 回复到达"

**状态变化**：
- `animationPhase`: `done` → `waiting_ai`
- `thinkingState`: `none` → `thinking`
- `showApprovalButtons`: `true` → `false`

### 4. 驳回

**流程**：
1. 用户点击"驳"按钮，填写驳回理由
2. 隐藏审批按钮和遮罩
3. 显示皇帝气泡，打字机效果显示驳回理由
4. 打字完成后等待 2 秒
5. 气泡淡出（1 秒）
6. 进入 `waiting_ai` 阶段，显示 `thinking.gif`
7. 后续流程同"AI 回复到达"

**状态变化**：
- `animationPhase`: `done` → `waiting_ai`
- `thinkingState`: `none` → `none` → `thinking`
- `bubbleVisible`: `false` → `true` → `false`
- `showApprovalButtons`: `true` → `false`

### 5. 结束对话

**流程**：
1. 用户点击"结"按钮
2. 隐藏审批按钮和遮罩
3. 显示皇帝气泡，打字机效果显示"此间事了，你先退下吧"
4. 打字完成后等待 3 秒
5. 气泡淡出（1 秒）
6. 清空任务列表，准备接收新任务

**状态变化**：
- `animationPhase`: `done` → `done`
- `bubbleVisible`: `false` → `true` → `false`
- `showApprovalButtons`: `true` → `false`
- `displayedTasks`: `[task]` → `[]`

## 关键状态说明

### animationPhase（动画阶段）
- `emperor_bubble`: 显示皇帝气泡
- `waiting_ai`: 等待 AI 回复，显示 thinking.gif
- `ai_ready`: AI 回复到达，显示 idea.gif
- `minister_bubble`: 显示大臣气泡
- `done`: 动画完成，显示审批按钮

### thinkingState（思考状态）
- `none`: 不显示 GIF
- `thinking`: 显示 thinking.gif
- `idea`: 显示 idea.gif

### bubbleVisible（气泡可见性）
- `true`: 气泡显示（opacity: 100）
- `false`: 气泡隐藏（opacity: 0）

## 时间参数

- 皇帝气泡打字速度：50ms/字符
- 皇帝气泡停留时间：2 秒
- 气泡淡出动画：1 秒
- idea.gif 显示时间：1 秒
- 大臣气泡打字速度：50ms/字符
- 大臣气泡停留时间：3 秒
- 审批按钮延迟：1 秒（气泡淡出后）

- `frontend/src/recourse/idea.gif` - 灵感来了动画

- GIF 尺寸：256x256 像素
